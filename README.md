Galleria Art Gallery
A High-End Digital Sanctuary for Art Enthusiasts
Galleria is a premium web platform designed to bridge the gap between classical artistry and modern web technology. It serves as a curated space for collectors to explore over 50+ masterpieces, from traditional portraits to cutting-edge abstract movements, all within a luxury dark-themed interface.

 Features
Dynamic Gallery: 50+ unique art cards with high-quality images.

Smart Filtering: Categorize art by Abstract, Modern, and Portrait.

Real-time Search: Instant search functionality to find masterpieces by title.

Advanced Pagination: Optimized loading with 8 items per page for better performance.

Persistent Cart & Wishlist: Integrated with LocalStorage to save your favorites even after a refresh.

Fully Responsive: Optimized for Mobile, Tablet, and Desktop using Bootstrap 5.

Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)
Frameworks: Bootstrap 5.3
Libraries: jQuery 3.7 (for DOM manipulation and logic)
Storage: Browser LocalStorage
